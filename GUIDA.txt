Come configurare Doomseeker per giocare a NIGHTMARE

Questa opzione è richiesta solo se è la prima volta che giocate
ad un gioco online con Doomseeker.

Scaricate i file di Nightmare MULTIPLAYER,
installateli ed entrate nella cartella di gioco,
avviate il file "Doomseeker.exe",
nella finestra di configurazione selezionate la voce "Zandronum",
nella sezione "Path to game executable" selezionate il file "NIGHTMARE_MP.exe",
premete su 'Apply' e 'OK'.
...

Come accedere ad un server di gioco attivo (solo NIGHTMARE)

Scegliere un server nella lista principale,
controllare la sezione "IWAD" di quel server ed accertarsi che ci sia scritto "NIGHTMARE.wad",
se queste due opzioni sono valide, allora vi basterà fare doppio click sul server scelto.

Se non si avvia il server, ricontrollare che il server abbia come file di gioco (IWAD) il file "NIGHTMARE.wad".
...

Creare una sessione di gioco online

Premere su 'File' in alto e selezionare 'Create game',
per configurare la sessione di gioco, vi lascio fare a voi.

Selezionare 'Start server',
attendere che il server si apra,
premere su 'File' e 'Join server'.
